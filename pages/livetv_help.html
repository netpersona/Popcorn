Live TV Setup Guide

How to integrate Popcorn with Plex's Live TV & DVR

Requirements: The Live TV feature requires a Plex Pass subscription. This feature allows your automated movie channels to appear as native Live TV channels in Plex.

What is Live TV?

Popcorn's Live TV feature emulates an HDHomeRun tuner device, allowing Plex to discover and integrate your automated movie channels as native Live TV channels. This provides a seamless viewing experience within Plex's Live TV & DVR interface.

Setup Instructions

Step 1: Enable Live TV in Popcorn

  1. Go to the Settings page
  2. Scroll to the "Live TV Integration" section
  3. Check the "Enable Live TV & DVR Integration" checkbox
  4. Click "Save Live TV Settings"

Step 2: Add Popcorn as a Live TV Source in Plex

  1. Open Plex and go to Settings
  2. Navigate to Live TV & DVR
  3. Click Set Up Plex DVR (or Add Device if you already have DVR configured)
  4. Plex should automatically detect Popcorn as an HDHomeRun device
    • Device name: Popcorn
    • Device ID: {{ device_id }}
  5. Select the Popcorn device and click Continue
  6. Plex will scan for channels - all your Popcorn channels should appear
  7. Select the channels you want and click Continue
  8. Configure the EPG (Electronic Program Guide):
    • When prompted for program guide data, select Have an XMLTV guide
    • Enter the XMLTV URL: {{ xmltv_url }}
  9. Complete the setup wizard

Step 3: Verify Your Channels

  1. In Plex, navigate to Live TV from the main menu
  2. You should see all your Popcorn channels listed with their current programming
  3. Click on a channel to start watching

Technical Details

Troubleshooting

Plex Doesn't Detect Popcorn

  • Ensure Live TV is enabled in Popcorn's settings
  • Verify that Plex and Popcorn are on the same network
  • Try manually entering the discovery URL: {{ base_url }}discover.json
  • Check that your Popcorn server is accessible from Plex

Channels Not Appearing

  • Verify that you have schedules configured for your channels
  • Check the lineup endpoint: View Lineup
  • Rescan channels in Plex's Live TV settings

Program Guide Not Showing

  • Verify the XMLTV URL is correct: {{ xmltv_url }}
  • Check the XMLTV feed: View XMLTV
  • Refresh the guide data in Plex's Live TV settings

Streaming Issues

  • Ensure FFmpeg is installed and working
  • Check that your Plex server has access to the movie files
  • Verify that the correct movie is scheduled for the current time
  • Check server logs for FFmpeg errors

How It Works

Popcorn's Live TV feature works by:

  1. Emulating an HDHomeRun device - Plex sees Popcorn as a hardware TV tuner
  2. Generating a channel lineup - Each Popcorn channel becomes a TV channel
  3. Creating an EPG (Electronic Program Guide) - 7 days of programming based on your schedules
  4. Streaming in real-time - When you tune to a channel, Popcorn:
    • Determines what movie should be playing based on the schedule
    • Calculates the offset (how far into the movie you should be)
    • Streams from that point using FFmpeg in MPEG-TS format
Note: The Live TV feature is disabled by default and requires a Plex Pass subscription. This is because Plex's Live TV & DVR features are only available to Plex Pass subscribers.
